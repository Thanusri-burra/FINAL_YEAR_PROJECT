🎒 Lost & Found AI

Intelligent system to locate lost objects using CCTV video analysis

🔍 What This Module Does

The Lost & Found AI module helps identify the exact location of misplaced objects by matching a lost item image with frames extracted from multiple CCTV videos installed across the campus.

🧠 AI Model Used

  • Pretrained CNN – ResNet50
  • Deep Feature Extraction
  • Cosine Similarity Matching
  • Transfer Learning (No dataset training)

⚙ How It Works

  1. User uploads a lost item image
  2. CCTV videos are processed offline
  3. Frames are extracted at intervals
  4. Features are compared using similarity
  5. Best matching camera and room detected

📥 Input & 📤 Output

  • Input: Lost item image + CCTV videos
  • Output: Camera ID, Room No, Timestamp, Confidence

📌 Why No Dataset Training?

Instead of training a custom dataset, a pretrained ResNet model is used. This reduces training time, computational cost, and improves generalization for unseen objects.
